Cruise Control

There's also cruise control running on this machine.

You too can add your tests.

  • All tests are simply placed into a /tests/ directory.
  • phpunit tests need an AllTests.php
  • Tests should run from your-package-in-svn/trunk. Like:
    clockwerx@clockwerx-desktop:~/pear-svn/packages/Crypt_RSA/trunk$ php tests/AllTests.php
    or clockwerx@clockwerx-desktop:~/pear-svn/packages/Crypt_RSA/trunk$ pear run-tests -r tests/
  • If your package need to be installed, you are doing it wrong. Fix the directory structure so it's like how the package is installed.
  • If your package needs an extension to be installed, you should make sure tests skip, not fail
